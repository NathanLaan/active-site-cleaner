# Privacy Policy - Clear Site Data Extension

## Data Collection

This extension does not collect, store, or transmit any personal data or browsing information.

## What the Extension Does

- Clears cookies and site data ONLY for the currently active website
- All operations are performed locally in your browser
- No data is sent to external servers
- No analytics or tracking is performed

## Permissions Used

The extension requires certain permissions to function:

- **cookies**: To read and remove cookies for the current site
- **browsingData**: To clear site storage (localStorage, IndexedDB, etc.)
- **activeTab**: To identify the current website domain
- **tabs**: To query information about the active tab
- **storage**: To clear browser storage for the site

## Data Storage

- The extension does not store any user data
- No settings or preferences are saved
- Each use is completely independent

## Third-Party Services

This extension does not communicate with any third-party services or external servers.

## Changes to This Policy

Any changes to this privacy policy will be reflected in extension updates.
